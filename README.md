# 🌦️ Weather App

A simple and responsive Weather Application that provides real-time weather information for any city, state, or country using a weather API.

## 🚀 Live Demo

🔗 https://weather-app-iota-puce-30.vercel.app

## 📌 Features

- Search weather by city name
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
- Clean and responsive UI
- Fast and easy to use

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
- Vercel (Deployment)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/NarayanSharma123/Weather-App.git
```

2. Navigate to the project directory

```bash
cd Weather-App
```

3. Open `index.html` in your browser.
